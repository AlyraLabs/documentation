## Example requests for `/v1/tools`

### Fetch all tools

```ts
import axios from "axios";

const response = await axios.get("https://api.alyra.finance/v1/tools");

console.log(response);
```

### Fetch tools filtered by chain IDs

```ts
import axios from "axios";

const response = await axios.get("https://api.alyra.finance/v1/tools", {
  params: { chains: "1,56,10" }
});

console.log(response);
```
